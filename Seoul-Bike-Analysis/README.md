📋 #Project Overview
This project was developed as part of the Microsoft Elevate AICTE Internship. The goal was to analyze urban mobility patterns and environmental factors to optimize bike-sharing supply in Seoul. By utilizing Power BI's built-in Machine Learning capabilities, I identified key drivers that fluctuate demand.

🚀 Key Insights (AI-Driven)
Using the Key Influencers AI model, I discovered the following:

*Peak Hour Surge: Demand increases by an average of +605.1 units during commuter peak hours.

*Weather Impact: Rising temperatures are a primary driver, with a +347.2 unit increase for every significant rise in average temperature.

*Environmental Resistance: Rainfall above 0mm correlates with a -575.9 unit drop in rentals, identifying windows for fleet maintenance.

🛠️ #Tech Stack & Tools
Power BI: Advanced Visualization, Key Influencers (AI), Decomposition Tree.

Power Query: Feature Engineering (Created Peak Hour, Day Name, and Month Name columns).

DAX: Developed measures for average environmental conditions and rental sums.

Design: Professional Glassmorphism-style UI/UX.
