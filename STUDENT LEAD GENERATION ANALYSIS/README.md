# Student Lead Generation Analysis
An interactive Power BI dashboard designed to analyze and optimize the student enrollment funnel through lead source tracking and demographic insights.

## 📊 Key Insights
* **Lead Source Performance**: Evaluated the effectiveness of various channels including Social Media, Website, and Referrals.
* **Demographic Analysis**: Visualized lead distribution across different cities and educational backgrounds.
* **Conversion Tracking**: Identified high-potential leads based on parent occupation and current education level.

## 🛠️ Technical Details
* **Data Source**: `leads_basic_details.csv` containing 1,300+ lead records.
* **Tool**: Power BI Desktop.
* **Key Visuals**: Slicers for city-based filtering, Lead Source distribution charts, and demographic tables.

## 📁 Files in this Folder
* `student data (1).pbix`: The primary Power BI dashboard file.
* `leads_basic_details.csv`: The raw dataset used for the analysis.
