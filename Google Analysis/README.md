Project Overview
This project analyzes 5 years of historical stock data for Google (GOOG) to identify patterns between price movements and trading volume.

Key Technical Skills
ETL & Data Cleaning: Used Power Query to transform raw CSV data, handle null values, and format date-time stamps for time-intelligence analysis.

DAX Formulas: Developed custom measures for 52-week High/Low and Price Momentum.

Interactive Visualization: Implemented a Date Slicer and Combo Charts to track daily price vs. volume trends.

Business Insight
Identified that Mondays typically account for the highest average trading volume at 21.59%, indicating high "Market Open" momentum.
